# Book-record-management
Book record management system using (C++)
 # Book Information
- Author Name
- Title of the book
- Year of publication
- Edition 
- Publishers name
- Cost of the book
- Books type
# Services to compute
- No. of years stayed (based on pub-year)
- Category of the book (based on title)
- Level of expense (based on cost)
- No of book category (based on book type)
